Что входит в SEO продвижение сайта

01

Семантическое ядро

  • Сбор базовых коммерческих и информационных запросов
  • Отбор ключевых фраз для сайта через Яндекс Wordstat и Google Keyword Planner
  • Парсинг поисковых подсказок
  • Сбор запросов с сайтов конкурентов
  • Удаление дублей, нулевых и нецелевых ключей
  • Анализ сезонности спроса
  • Кластеризация семантического ядра
  • Проектирование структуры с учетом кластеризации
02

Техническая оптимизация

  • Настройка robots.txt
  • Переход на человекопонятные ссылки
  • Склеивание зеркал сайта
  • Настройка пагинации
  • Обнаружение и удаление битых ссылок
  • Сжатие изображений
  • Внедрение микроразметки Schema
  • Обнаружение и удаление дублей title и description
  • Переход на https протокол
  • Создание sitemap.xml — карты сайта
03

Юзабилити

  • Создание уникальной страницы 404
  • Внедрение breadcrumbs — хлебных крошек
  • Оптимизация скорости загрузки
  • Проверка кроссбраузерности
  • Внедрение адаптивной версии для мобильных
  • Удаление всплывающих окон
  • Корректировка навигации по разделам
  • Проверка корректности работы поиска
  • Упрощение форм лидогенерации
  • Внедрение кнопки «Заказать в один клик»
  • Перелинковка страниц
